- 🔄 Revise core JavaScript concepts
- 🧪 Practice coding exercises and examples
- 📝 Document important notes and takeaways
- 🛠️ Build small projects to apply what I learn

## 🛠️ How to Use

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/javascript.git
   cd javascript
   ```

2. **Explore the folders** — Each folder contains examples and notes on a specific topic.

3. **Run the code** — Open any `.js` file and run it using Node.js:
   ```bash
   node basics/variables.js
   ```

4. **Practice** — Modify the code, experiment, and learn by doing! 💡
